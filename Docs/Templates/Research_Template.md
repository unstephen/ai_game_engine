### 5.2 接口设计

```cpp

public:
    bool Initialize(const VTDesc& desc);
    void Update(float deltaTime);

    // ...

```

### 5.3 数据结构

```cpp
struct VTPage {
    uint32_t pageId;
    uint32_t physicalIndex;
    uint32_t lastAccessTime;
    // ...
};
```

### 5.4 关键流程

```
1. 应用请求纹理采样
       ↓
2. 查询虚拟→物理映射
       ↓
3. 未命中 → 触发页面加载
       ↓
4. GPU 上传 → 更新映射表
       ↓
5. 返回物理纹理句柄
```
