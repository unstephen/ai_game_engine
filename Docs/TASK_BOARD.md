## 🎯 认领任务流程

### 步骤 1: 选择任务
- 查看 [待分配] 的任务
- 评估难度和自己的时间
- 确认感兴趣的方向

### 步骤 2: 创建 Issue
- 进入 GitHub Issues
- 新建 Issue，标题：`[任务认领] RHI-001 - 虚拟纹理调研`
- 描述中说明：
  - 预计完成时间
  - 调研方向
  - 产出形式（文档/代码/PPT）

### 步骤 3: 开始工作
- 创建分支：`feature/RHI-001-virtual-texture`
- 定期提交进度
- 遇到问题及时沟通

### 步骤 4: 提交 PR
- 完成所有要求
- 关联 Issue
- 等待代码审查

### 步骤 5: 合并与归档
- 通过审查后合并到 dev
- 更新任务看板状态
- 归档到已完成
